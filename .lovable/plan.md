

# SWIFT — Smart Web Intelligence Fast Trading

An agentic commerce platform with an orange/burnt orange monochrome UI, featuring AI-powered shopping that understands intent and handles everything from discovery to checkout.

## Pages & Navigation

### 1. Homepage / Landing Page
- Hero section explaining SWIFT's value proposition: "Describe what you need. We handle the rest."
- How it works (3 steps): Describe → AI Shops → One-Click Checkout
- Feature highlights (multi-retailer, combined cart, smart ranking)
- CTA to start shopping with the AI agent

### 2. AI Shopping Agent (Core Feature — Hackathon Host Kit)
- Conversational chat interface powered by Lovable AI
- Agent captures: budget, deadline, preferences, must-haves
- Outputs a structured shopping spec
- Multi-retailer product discovery from 3+ mock retailers (e.g., Amazon, Walmart, Party City) with realistic product data
- Transparent ranking engine scoring by: cost, delivery, preference match, set coherence
- "Why #1?" explanation for each recommendation
- Combined cart view showing items from multiple retailers with per-item cost/delivery
- Users can swap items and agent adapts

### 3. Combined Cart & Checkout
- Cart aggregating items across retailers with totals
- Simulated checkout orchestration: enter payment + address once, agent fans out per retailer
- Payment popup with options: Mobile Transfer (OPay, Kuda, Moniepoint), Bank Transfer, Card Payment
- Checkout confirmation with order summary
